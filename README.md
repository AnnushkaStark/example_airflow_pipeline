# Пример ETL пайплайна с оркестрацией

Для получения данных использовано api 

https://www.exchangerate-api.com

Этот проект является примером
 работы отркестратора с ETL процессом для реализации получения данных именно из этого API оркестрация и вся остальная логика избыточна


## Запуск 

1 .  В коренвой директории создать файл .env

2 . Заполнить переменные окружения из .env.template

3 . Запускать из корневой директории командой

```
docker-compose up --build
```
